# weathering_psv
### 移植自安卓版本的《挂机工厂》
#### PSV操作：

| 游戏操作 | PSV按键        |
| :------- |:------------- |
| 人物移动 | 方向键、左摇杆 |
| 选择地块 | 右摇杆         |
| 操作地块 | ×键            |
| 打开仓库 | □键            |
| 查看产出 | △键           |
| 锤子     | 右肩键         |
| 磁铁     | 左肩键         |
| 设置     | start键        |
| 人物状态 | select键       |

| UI操作 | PSV按键            |
| :------- |:--------------- |
| 按键选择 | 方向键、左摇杆   |
| 滑动条   | 方向键左右、肩键 |
| 确定     | ×键             |
| 返回     | ○键             |
