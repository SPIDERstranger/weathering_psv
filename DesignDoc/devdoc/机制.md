# 建筑参数

* 建造来源。如：果园由农场升级
* 建造地块。如：农田必须建造在草原上
* 建造成本。如：建造农田需要背包50食材
* 建造条件。如：农田地块旁边必须有水源
* 资源噪声。气候、资源、

# 常见启用操作

* 双方建立联系。自己起点，他人终点
* 连续联系：自己起点，他人终点。他人其他，再他人终点。如：三连路机制
* 连续联系：沿着道路走，直到岔路口。如果起点是道路，还需选方向
